1. Clonar el repositorio
2. Configurar .env
BASE_URL=http://localhost/admiflow/
# Database configuration
DB_HOST=localhost          # Database host
DB_NAME=admif                    # Database name
DB_USER=root                      # Database user
DB_PASS=1234                  # Database password
DB_TRUST_SERVER_CERTIFICATE=true # Trust server certificate (true/false)

# Database type: choose between 'sqlsrv' for SQL Server or 'mysql' for MySQL
DB_TYPE=mysql

3. Importar la base de datos
4. Ejecutar en entorno local (XAMPP/Laragon)
